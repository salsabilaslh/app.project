# Quotes Data Analysis System

This project is a midterm assignment that demonstrates a full-stack workflow, including data collection, storage, backend API, and user interface.

The system collects quotes from a website, stores them in a database, and provides features for viewing, analyzing, and translating the data.

---

## Features

- View quotes with author information  
- Word count analysis  
- Translation to Indonesian and Korean  
- Web-based user interface  

---

## Tech Stack

- Python  
- BeautifulSoup (Web Scraping)  
- SQLite (Database)  
- FastAPI (Backend API)  
- Gradio (User Interface)
